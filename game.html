Slot Machine

Slot Machine

About the game:

This is a slot machine in its most basic version. The principle is simple: 

When button "SPIN" is pressed it starts to turn from left to right, the rotation time it's random each spin, between two and four seconds, when this time is up the machine starts to stop again from left to right.  At this point, if there is some winning patern, points are added to your score. 

How to win score:

You win points if there are specific types of matches (paterns) between the same figure, depending of the matched figure and the type of match, you recieve more or less points.

There are two tipes of winning paterns:

Basic paterns:

If there is the same figure minimum three times in a row, starting from the left column, you win points. The score is different depending of the number of repetitions (three, four or five).  


Special paterns:

Not only with rows is how you gain points, furthermore, there are special paterns that will give you  points. Again, it has to match the same figure starting from the left column, but this time forming the next paterns:
